# DataCompare —— Excel vs PDF 年度报告数值比对工具

> Ai Coding 项目（含 AI 驱动的数据分析模块）

---

## 项目概述

DataCompare 是一款面向审计/财务/统计领域的数据核对工具，核心解决 **Excel 指标表与 PDF 年度报告之间的数值自动比对** 问题。用户上传 Excel 文件和 PDF 报告后，系统自动完成 PDF 文本提取、数值定位、模糊匹配、置信度计算、核对状态管理及报告导出的全流程自动化。

## 核心功能

### 1. Excel 智能解析
- **双格式自动检测**：支持"简单格式"（指标名+数值列）和"宽表格式"（分类列+年份列）的自动识别
- **单元格颜色提取**：利用 win32com 提取原始单元格背景色，支持颜色→来源类型（年鉴/司局报告/AI检索）的用户自定义映射

### 2. PDF 文本与数值提取
- **逐页文本提取**：基于 pdfplumber 的 PDF 文本抽取
- **全量数值提取**：支持千分位、空格分隔、括号负数、百分号等多种数值格式的正则匹配
- **单位检测**：自动检测中文单位（万元/亿元/百万元/千元/元/%）并与指标单位做换算比对
- **页码映射**：通过 char_offset 实现字符位置→页码的反查

### 3. 核心比对引擎 —— 比对与置信度判定
- **来源解析**：从 Excel 的 source_file 字段解析出多个数据来源（年鉴/报告/URL），支持 `+P页码` 语法
- **PDF 文件名模糊匹配**：四级匹配策略（精确→包含→前缀→年份）
- **上下文窗口搜索**：在指标来源指定的页码中搜索目标数值，提取前后各 50 字符的上下文
- **三层置信度模型**：
  - `模糊匹配分数（权重 60%）`：基于 rapidfuzz 的指标名称与上下文相似度
  - `距离分数（权重 25%）`：关键词与数值在上下文中的距离
  - `单位一致性分数（权重 15%）`：PDF 单位与 Excel 单位是否匹配
- **单位换算比对**：支持中文单位的自动换算，容差可配置（默认 ±0.01）

### 4. **AI 分析模块** (Ai Coding 部分)
> 集成 DeepSeek 大语言模型实现智能数据分析

- **智能数据验证**：对于 URL 来源的指标，调用 DeepSeek API 分析网页文本中是否存在对应数据
- **层级化分析**：基于一级→二级→三级标题→数据项的完整分类层级进行精确匹配
- **运算逻辑推断**：如果数据需要计算得出（求和/均值/百分比等），AI 自动给出运算逻辑和详细步骤
- **置信度分级**：返回 high/medium/low 三级置信度判断
- **结构化输出**：JSON 格式的标准化分析结果，含匹配文本、数值、运算逻辑与解释

### 5. **请求控制器** (Ai Coding 部分)
> 与 AI 分析模块配合，为 URL 数据采集提供请求控制能力

- **域名级别令牌桶限流**：每个域名独立控制请求速率，支持突发（burst）容量和最小间隔
- **指数退避重试**：带随机抖动的重试策略，可配置重试次数（默认 3 次）、初始延迟、最大延迟
- **并发控制**：基于 asyncio.Semaphore 的全局并发数控制
- **URL 数据采集**：自动处理编码检测（Content-Type / HTML meta）、HTML 正文提取（BeautifulSoup）、非正文元素过滤
- **批量请求**：支持带进度回调的批量 URL 抓取，缓存已获取的页面
- **完整统计**：请求数/成功率/重试次数/限流次数/等待时间的详细统计

### 6. 交互式核对工作台
- **左右分栏布局**：左侧指标列表（支持搜索、年份筛选、状态着色）+ 右侧 PDF 阅读器与匹配结果面板
- **PDF 浏览器内渲染**：使用 react-pdf + pdfjs-dist 在浏览器中直接渲染 PDF 页面
- **智能高亮**：最高置信度匹配的数值在 PDF 页面中自动高亮（含跨 span 的 MutationObserver 方案）
- **手动绑定**：用户可在 PDF 上划选文本，手动绑定到当前指标
- **键盘快捷键**：上下切换指标、Enter 确认/Space 标记争议、PageUp/Down 翻页、Ctrl+Z 撤销、Ctrl+E 导出

### 7. 报告导出
- **汇总报告**：多 Sheet 的 Excel 报告（汇总对照表 + 各 PDF 详情 + 未找到项）
- **标色原表**：在原 Excel 基础上按核对状态（已核对/未核对）自动标色
- **纯文本导出**：PDF 提取的纯文本文件下载

## 项目亮点总结

| 维度 | 说明 |
|------|------|
| **业务价值** | 将审计/财务对账从纯人工操作提升为半自动化流程，单次核对时间从数小时压缩到分钟级 |
| **技术深度** | FastAPI 异步架构 + 缓存机制（内存+磁盘二级缓存）+ PDF 渲染引擎集成 + 模糊匹配算法 + 单位换算体系 |
| **AI 赋能** | (Ai Coding) 集成 DeepSeek 大模型实现 URL 来源数据的智能验证；基于令牌桶限流+指数退避重试的请求控制器，为 AI 数据采集提供支持 |
| **交互体验** | 键盘全快捷键操作 + 实时进度追踪 + 多层级高亮 + 手动绑定兜底机制，兼顾效率与灵活性 |
| **工程规范** | Pydantic v2 全栈模型统一、类型提示覆盖、异步并发控制、PDF 预处理加速、错误处理与日志 |
